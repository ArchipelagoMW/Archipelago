# Celeste (Open World) Guía de Instalación

## Programas necesarios
- Celeste en su última versión oficial disponible para PC (Versión 1.4) 
- Olympus (El Manager de Mods de Celeste) lo puedes descargar por este [enlace](https://everestapi.github.io/)
- La versión más reciente del mod Archipelago Open World desde su [página de GitHub](https://github.com/PoryGoneDev/Celeste-Archipelago-Open-World/releases)

## Pasos para su instalacón (Windows/Linux)

1. Instala la versión más reciente de Celeste para PC (versión 1.4)
2. Instala `Olympus` (el administrador de mods) y `Everest` (el que ejecuta los mods) siguiendo las instrucciones:
[Instrucciones de configuración de Olympus](https://everestapi.github.io/)
3. Coloca el archivo `Archipelago_Open_World.zip` que descargaste desde su página de GitHub en la carpeta llamada
`mods`  que se encuentra en la carpeta raíz donde se instaló Celeste.
4. (Recomendado) Desde el menú principal, entra en `Mod Options` y activa `Debug Mode` para `Everest` o `Always`.
Esto te dará acceso a un Cliente de Texto rudimentario que podrás abrir oprimiendo la tecla `~`.

## Unirte a una partida

1. Abre Everest desde el programa de Olympus con el mod de Archipelago Open World activado
2. Selecciona la opción `Connect` del menú principal, esto abrirá las opciones para conectarse a la partida
3. Usa libremente el teclado para ingresar los datos para conectarse: tu nombre de usuario, el servidor y la contraseña
4. Una vez conectado, puedes usar el Menu Debug (el que se abre con `~`) como un Cliente de Texto, al escribir "`!ap`"
seguido de lo que escribirías en un Cliente de Texto normal.
