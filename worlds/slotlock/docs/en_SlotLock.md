# SlotLock

## Slotlock has:
* Client
* Ability to send items when finding a locked World / or send those items as starter items
* Bonus Item Packages that contain a definite amount of items when found
* A way to "Goal" the world

## Both have:
* Random starting unlocked slots
* A list of worlds that are locked
* The ability to include every worlds as unlockable except those listed
* Now compatible with prefilled worlds like OoT

## Shahrazad has:
* No Client
* No Checks added
* Prehinted items for world unlocks
