# Final Fantasy 12 Open World Start Guide

## Requires the steam version of Final Fantasy XII: The Zodiac Age

Download the following:

* ff12_open_world.apworld
* FF12Randomizer#.#.#.#.7z

# AP World and Seed Setup

* Double click the ff12_open_world.apworld to install the AP world
* Run the Archipelago Launcher (ArchipelagoLauncher.exe)
* Click Generate Template Options to open the folder with the template yaml files
* Copy the Final Fantasy 12 Open World.yaml template to the Players folder
* Open your YAML file and edit the name: Player{number} to set your name.
* {number} is needed if there will others in the seed with a similar name, you can remove it otherwise.
* Modify any other options you want further below in the YAML.
* By default, locations are very limited for where progression/useful items can appear. Turn on any include options you want.
* If hosting: Run the Archipelago Launcher (ArchipelagoLauncher.exe) and Click Generate

# Randomizer Seed Setup After Room is Opened

* In the AP room, click Download Patch File... to download your .apff12ow file
* Extract FF12Randomizer#.#.#.#.7z (using Windows, 7zip, etc)
* Open the extracted folder and run FF12Rando.exe
* It may take a while at first launch.
* On the 1. Setup page, download and install the required mods and any optional mods (Requires a NexusMods account)
* At the bottom of the 1. Setup page, click Load from ZIP/APFF12OW and select your player .apff12ow file from step 1
* After you do this, the randomizer should switch to 'Archipelago Mode'
* On the 2. Flags page, select any additional settings
* On the 4. Finish page, click Generate Seed

# AP FF12 Open World Client

* Launch the game in ENGLISH (Only language supported)
* Run the Archipelago Launcher (ArchipelagoLauncher.exe)
* Click FF12 Open World Client
* Connect to your room with the correct port and player name

WARNING: Loading any other saves while connected to the client may send some unexpected checks to the server. Be careful not load any other saves (The load menu in this game is funky sometimes). I'm still working out the best way to avoid this.
