# Osu! Multiworld Setup

short non detailed version of how to play osu archipelago:
* Setup oauth application on osu website https://osu.ppy.sh/home/account/edit
* Open osu!client in archipelago launcher
* Run the following commands with the API keys and stuff from step 1`/set_client_id`, `/set_api_key`, and `/set_player_id`
* run `/save_keys` to save step 3 for future use, use `/load_keys` in the future to skip step 3
* Connect to AP
* `/auto_track all`
* `/auto_download`
* `/save_settings` (use `/load_settings` in the future to skip step 6 and 7)
* `/download next` to get the next song in logic (first song in this case)
* Play the first song and let the osu!client send check and download the next songs automatically
* Repeat until victory
