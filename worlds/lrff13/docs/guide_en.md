# Final Fantasy 12 Open World Start Guide

## Download the following:

* lrff13.apworld
* LRRandomizer#.#.#.#.7z
* [Nova Chrysalia mod manager v2.0.3](https://mega.nz/file/24gyiB7b#nTIlktJb8ZCo8dXZDxCgwsdHAUDPURQhixSfucypIVg)

## AP World and Seed Setup

* Double click the lrff13.apworld to install the custom world
* Run the Archipelago Launcher (ArchipelagoLauncher.exe)
* Click Generate Template Options to open the folder with the template yaml files
* Copy the Lightning Returns Final Fantasy XIII.yaml template to the Players folder
* Open your YAML file and edit the name: Player{number} to set your name.
* {number} is needed if there will others in the seed with a similar name, you can remove it otherwise.
* Modify any other options you want below.
* If hosting: Run the Archipelago Launcher (ArchipelagoLauncher.exe)
* Click Generate
* Go the output folder, and you will find your created AP_#####.zip file.
* To host the game, upload AP_#####.zip to: https://archipelago.gg/uploads

## Randomizer Seed Setups

* Extract the Nova Chrysalia.7z to a folder of your choosing (using Windows, 7zip, etc).
* Run Nova Chrysalia for the first time and select under Global Installation: "Unpack Game Data (EN Audio)" or "Unpack Game Data (JP Audio)" for the language audio you prefer
* Note: You may need to launch LR normally from Steam first
* Once the AP room is created, click the "Download Patch File" link to download a .aplrff13 mod file.
* Extract LRRandomizer#.#.#.#.7z to a folder of your choosing.
* Open the extracted folder and run LRRando.exe
* On the 1. Setup page and at the top, setup the paths to your Steam LR FFXIII folder and Nova.
* At the bottom of the 1. Setup page, click Load from ZIP/APLRFF13 and select your .aplrff13 file
* After you do this, the randomizer should switch to 'Archipelago Mode'
* On the 2. Flags page, select any additional settings
* On the 3. Finish page, click Generate Modpack
* Open Nova Chrysalia and install the generated .ncmp modpack. It will appear in the packs folder of LRRandomizer
* Launch the game from Nova Chrysalia in "Unpacked" mode with the "Voice Over Settings" matching the language you unpacked game data for.

## AP LR FF13 Client

* Launch the game
* Run the Archipelago Launcher (ArchipelagoLauncher.exe)
* Click Lightning Returns Final Fantasy XIII Client
* Connect to your room with the correct port and player name

WARNING: Loading any other saves while connected to the client may send some unexpected checks to the server. Be careful not load any other saves.
